# GoogleCloudPlatform 

Purpose for project is to put in one place useful commands and small examples for GCP

## Prerequisites

You need to:

1. have [GCP SDK](https://cloud.google.com/sdk) installed.
2. login into your GCP:

```
gcloud auth login
```

* [AutoML Vision](docs/AutoML.md)
